Stage de 1ère année : DSI de l'Université de Rouen

5 Semaines (Juin 2025)
Mont-Saint-Aignan
Équipe Développement (20 pers.)

Contexte

La DSI gère l'ensemble du système d'information de l'université (réseaux, maintenance, développement).

Organisation : Travail en binôme dans un bureau partagé. Communication via Rocket.chat et supervision par mon tuteur (Robin).

Environnement Tech

J'ai travaillé sur une stack technique moderne et robuste :

Java Spring Boot Thymeleaf JavaScript PostgreSQL GitLab IntelliJ IDEA

Projets et missions : Application "Historique-RH"

Objectif : Améliorer l'ergonomie de l'application interne permettant de consulter l'historique des employés.

  • Correction de bugs : Analyse et résolution d'incidents front et back-end.
  • Feature "Autocomplétion" : Développement d'un système de suggestion automatique lors de la saisie (recherche).
    Technique : Création d'un Controller Java pour interroger la BDD et adaptation du JavaScript pour afficher 5-6 suggestions dynamiques.
  • Réunions : Participation aux points d'équipe sur l'authentification 2FA.
